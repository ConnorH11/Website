Incus Deployment with OpenTofu

Automating container deployment, user provisioning, and networking.

View on GitHub

This project uses OpenTofu to automate the deployment of Incus containers. It includes deploying Ubuntu containers, provisioning users with passwordless sudo access, injecting SSH keys from GitHub, and joining containers to a Tailscale network.
